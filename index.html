Presentation du Bot

Travailler efficacement avec nos commandes discord

Le bot discord s'adresse principalement à un groupe de travail en distanciel avec l'utilisation du logiciel discord. Ces commandes sont indispensables pour travailler dans des conditions optimales.

Presentation des fonctionnalités du Bot

  • Rappel mail
  • Mute vote
  • Alea
  • Ping loop
  • Agenda
  • Clear

Listes des commandes :

  1. !Rappel : permet d'envoyer un mail avec le nom d'user et l'intitulé du message

  2. !Mute : permet de muter une personne en procedant à un vote

  3. !Alea : permet de choisir aléatoirement une personne

  4. !Ping : permet de spam un nombre defini de message

  5. !Agenda : permet d'afficher le prochain jour en distanciel

  6. !Clear : permet de clear le tchat
